## Author Keegan Trevillian

The following is a leaflet map that depicts the war in Ukraine using data from 2023.
The data was collected via The World Bank Group. I was able to trim out excess data so that I could display just the 
fatalities recorded. I then converted it into a geoJSON in which I created the leaflet map that can cycle through the year.
Each month has a dynamic label that allows for visualizing when the conflict was at its peak. 
The user is able to navigate the map both by clicking arrows or by dragging a slider bar.
